Session Domain Driven Design Mercredi 11 Septembre 2019

Format : Forum ouvert 34 participants (avec Greg et Adam) What is a bounded context? Is it the same to speak Canadian in France or in Canada? When a word ain’t the same for different people, they don’t need the same software to be built. Domain models belong in a context, we solve the problem in a bounded context. It removes complexity. When adding a new concept that encompass all bounded context will create a new system.
