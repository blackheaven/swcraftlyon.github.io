Opération invalide sur les aggrégats

“Ça dépend”, du métier et du language. Les code retours permettent de suivre le chemin logique. Les exceptions court-circuitent la logique et permettent de se concentrer sur les cas qui marchent. On peut demander au métier ce qui est attendu quand l’exception est lancé.

Le métier n’est pas infini

Le métier a des contraintes, les types primitives ne conviennent pas on peut utiliser les types pour contraindre le métier. On utilise des exceptions en production pour gagner en efficacité. Compilation > Runtime
